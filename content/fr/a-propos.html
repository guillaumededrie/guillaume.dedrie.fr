Ce qu'il faut savoir sur ce site Web

Ce site a été développé sous Licence Creative Commons Attribution 3.0 non transposé sauf mention contraire indiqué dans la page web.

Il utilise les technologies suivantes :

  • Hyde pour la génération du site statique
  • Initializr un ensemble d'outils pour l'architecture des pages web :
    • Bootstrap framework CSS propulsé par Twitter.
    • Modernizr une librairie Javascript pour HTML5/CSS3.
    • jQuery un framework javascript.

Au niveau de l'hebergement, j'utilise :

  • GitHub pour l'hébergement du code source (disponible ici sous la MIT licence).
  • OVH pour l'hébergement du serveur.
  • Gandi pour l'hébergement du nom de domaine.
